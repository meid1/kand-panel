VPN-панель, которая ставится одной командой

xray-core (VLESS-Reality) с умной маршрутизацией, Telegram-ботом и кабинетом, оплатами РУ-рынка и франшизами. Ноды подключаются без SSH — одной строкой.

⚡ Установить за 2 минуты 👀 Посмотреть демо
На чистом сервере (Ubuntu/Debian)
curl -fsSL https://raw.githubusercontent.com/meid1/kand-panel/main/install.sh | bash

Всё для VPN-сервиса из коробки

Не просто раздача ключей — готовый продукт: маршрутизация, бот, оплаты, аналитика, франшизы.

🇷🇺

Умная маршрутизация

РФ-сервисы напрямую, остальное через VPN. YouTube без рекламы. Свои сайты — в админке.

🖥

Ноды одной командой

Сервер сам ставит xray + агент и подключается по mTLS+JWT. Без ручного SSH.

📊

Здоровье бизнеса

Выручка, ARPPU, LTV, отток, воронка, прогноз дохода, графики — как в аналитике, а не «сколько ключей».

🤖

Telegram-бот и кабинет

Подписка, оплата, триал, автопродление; веб-кабинет с дип-линками Happ/v2rayNG и QR. Тексты правятся в вебе.

💳

7 платёжек РУ-рынка

ЮKassa, Platega, RollyPay, Wata, Lava, CryptoBot, Cryptomus. Автопродление с баланса без хранения карт.

🏷

Франшизы

Свой бренд/бот/домен, изоляция клиентов, учёт доли и выплат. Один движок — много брендов.

🛡

Лимит на «обход»

Учёт ГБ, докупка/списание, авто-стоп при исчерпании — метринг встроен в панель.

🔁

Миграция

Импорт с 3x-ui / Marzban / Remnawave готовыми пресетами и любой базы по маппингу.

🔒

Безопасность

mTLS нод, аудит действий, rate-limit, проверка подписей платёжек, вход по паролю → JWT.

Как это выглядит

Данные на скриншотах — демонстрационные.

Чем отличается

от Remnawave / Marzban / 3x-ui
