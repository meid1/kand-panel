Конфигуратор установки

Выберите домен и нужные возможности — получите готовую команду. Вставьте её по SSH на свой сервер (Ubuntu/Debian), и панель установится и настроится сама. Пароль сервера вводить не нужно.

1 Домен и HTTPS

Домен, который A-записью указывает на ваш сервер. Оставьте пустым — поставится по IP (HTTPS тогда недоступен).
${chk}
Авто-HTTPS (Let's Encrypt)
Автоматический бесплатный сертификат — нужен домен выше

2 Протоколы xray

Какие протоколы отдавать клиентам. Reality — рекомендуется.

3 Возможности панели

Включите только нужное — выключенное не грузит сервер и не мешает клиентам. Всё можно поменять позже в панели.
Команда установки — вставьте на сервер по SSH
curl -fsSL https://get.kandpanel.com/install.sh | bash
После установки команда выведет адрес панели, логин и пароль. Нет своего сервера или хотите «под ключ»? Напишите @marius_support — поможем бесплатно.
